Release history

1.0-α4 (April 4th, 2009)

CSPM compiler:

  • comments
  • nametype statement
  • datatype statement
  • dotted values
  • data-carrying channels

CSP₀ support for:

  • Operators:
    • replicated internal choice (⊓)

1.0-α3 (November 21st, 2008)

Introduced CSPM compiler.

Many language features are implemented. The following aren't:

  • sequence comprehensions
  • set comprehensions
  • dotted channels
  • special definitions
  • comments
  • type definitions
  • include files

Also, many process operators are not yet supported, since the underlying CSP₀ library also doesn't support them.

1.0-α2 (July 6th, 2008)

CSP₀ support for:

  • Failures refinement
  • Operators:
    • replicated external choice (□)
    • timeout (▹, was incorrectly labeled interrupt previously)

1.0-α1 (February 24th, 2008)

CSP₀ support for:

  • Traces refinement
  • Operators:
    • prefix (→)
    • external choice (□)
    • internal choice (⊓)
    • interrupt (▵)
    • sequential composition (;)
    • interleaving (|||)
    • alphabetized parallel ([αP‖αQ])
    • interface parallel (〚α〛)
    • hiding (\)
    • renaming (〚μ〛)
